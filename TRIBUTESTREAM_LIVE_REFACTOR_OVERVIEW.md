# Tributestream Live - Platform Refactor Overview

**Date**: November 4, 2025  
**Version**: 1.0  
**Status**: Planning Phase

## Executive Summary

Tributestream is evolving from a funeral/memorial-specific livestreaming service to **Tributestream Live** - a comprehensive personal livestream library platform for life's important moments with integrated fundraising capabilities.

## Vision Statement

> "Your personal livestream library for life's most important moments - celebrate, share, and support causes that matter."

## Core Transformation

### FROM: Memorial-Focused Platform
- Primary audience: Funeral homes and grieving families
- Use case: Memorial services and funeral livestreaming
- Limited scope: End-of-life events only
- Somber, memorial-specific branding

### TO: Life Events Livestreaming Platform
- Primary audience: Individuals celebrating any important moment
- Use cases: Weddings, birthdays, graduations, memorials, fundraisers, community events
- Broad scope: All life's important moments worth sharing
- Celebratory, versatile, professional branding

## Key Platform Features

### 1. Personal Livestream Library
- Each user maintains their own collection of livestreamed events
- Individual dedicated pages for each stream/event
- Chronological archive of past events
- Upcoming events dashboard
- Share events with specific audiences

### 2. Dual Streaming Options
- **DIY Streaming**: Stream directly from phone/browser (WHIP)
- **Professional Service**: Hire Tributestream team for on-site production
- Flexible pricing tiers for both options

### 3. Integrated Fundraising System
- Donation capabilities on every stream
- Fundraising goals and progress tracking
- Multiple cause types:
  - Personal needs (medical, education, emergencies)
  - Memorial funds (funeral expenses, scholarships)
  - Charitable causes (nonprofits, community projects)
  - Celebration contributions (gifts, group presents)
- Real-time donation displays during livestreams
- Secure payment processing via Stripe
- Payout system for event organizers

### 4. Event Types Expansion
- **Celebrations**: Weddings, birthdays, anniversaries, graduations
- **Family Events**: Reunions, baby showers, milestone celebrations
- **Memorial Services**: Funerals, celebrations of life, memorial services
- **Fundraisers**: Charity events, benefit concerts, awareness campaigns
- **Community Events**: Town halls, community gatherings, local events
- **Professional Events**: Conferences, workshops, panel discussions

## Target Audiences

### Primary Users
1. **Individual Event Organizers**: People planning personal celebrations
2. **Families**: Connecting distant relatives for important moments
3. **Nonprofit Organizations**: Fundraising events and awareness campaigns
4. **Community Leaders**: Public events and town halls
5. **Professional Event Planners**: Hybrid event solutions

### Service Tiers
1. **Basic DIY**: Self-service phone streaming
2. **Premium DIY**: Enhanced features, multiple cameras, analytics
3. **Professional Service**: On-site team, multi-camera production
4. **Enterprise**: Custom solutions for organizations

## Strategic Goals

### Business Objectives
- **Expand market**: Beyond funeral industry to all life events
- **Increase recurring revenue**: Users creating multiple streams over time
- **New revenue stream**: Transaction fees on donations/fundraising
- **Professional services**: Premium on-site streaming services

### User Experience Goals
- **Lower barrier to entry**: Easy DIY streaming from phones
- **Build personal libraries**: Encourage users to document multiple life events
- **Community building**: Connect distant loved ones for important moments
- **Social impact**: Enable fundraising for meaningful causes

### Technical Goals
- **Scalability**: Support high concurrent stream volumes
- **Reliability**: 99.9% uptime for important life moments
- **Mobile-first**: Optimized phone streaming experience
- **Payment processing**: Secure, compliant donation handling
